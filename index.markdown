# KaaS
__KaaS__ stands for __[Karate](https://intuit.github.io/karate/) as a Service__ and it is exactly what this tool intends to do, it provides to its users the ability to run a [Karate feature](https://intuit.github.io/karate/#features) from the browser.

## Where can i find it ? 
Everything is hosted on github among three repositories : 
- [KaaS-ui](https://github.com/areguig/kaas-ui/)
- [KaaS-api](https://github.com/areguig/kaas/)
- [KaaS-doc](https://github.com/areguig/kaas-doc/)
